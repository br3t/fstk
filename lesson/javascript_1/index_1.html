Основы JavaScript 1.1. Операторы

В зависимости от количества операндов различают унарные, бинарные и тернарный операторы.

Операторы сравнения

  • (Не)равенство: ==, !=
  • Строгое (не)равенство: ===, !==
  • Больше/меньше: >, >=, <, <=

Арифметические операторы

  • Основные операции: +, -, *, /, %
  • Инкремент и декремент: ++, --

Строковые операторы

  • Конкатенации строк: +
  • Конкатенация с присвоением: +=

Побитовые операторы

  • &, |, ^, ~, <<, >>, >>>

Операторы присваивания

  • =, +=, -=, *=, /=, >>=, <<=, >>>=, &=, |=, ^=

Логические операторы

  • Логическое И: &&
  • Логическое ИЛИ: ||
  • Логическое НЕ: !

Условный оператор

  • условие ? если_условие_верно : если условие_ложно
