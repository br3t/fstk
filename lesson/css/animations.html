Анимации CSS

Код создания анимации

animation:animation-name animation-duration [animation-timing-function:ease] [animation-delay:0] [animation-iteration-count:1] [animation-direction:normal] [animation-fill-mode:none] [animation-play-state:running] 
div {
	background: #F00;
	margin: 30px 0 0;
	animation: myanimation 3s infinite;
}
@keyframes myanimation {
	100% {
		background: #00F;
		margin-left: 300px;
	}
}
