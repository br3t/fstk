Задача №1

Создать страницу курса криптовалют. На странице два select с выбором криптовалюты, при изменений значений получать с помощью ajax их курс относительно друг друга с https://www.cryptocompare.com и выводить на страницу. В списке валют: US Dollars, Bitcoin, Ethereum, Litecoin, Dogecoin. Использовать jQuery.
