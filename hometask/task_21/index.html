Домашнее задание по HTML5. Часть 3

Задача №1

Написать программу, которая принимает данные о температуре в течение недели в формате JSON и строит линейный график по ним. По оси Х - дни недели, по оси У - температура в градусах. При наведении на точку графика, соответсвующую конкретному дню недели, поялвяется подсказка с информацией о дне недели и температурой в это день. Если данные переданы для нескольких недель, то появляется селект для выбора недели. Использовать SVG.

Задача №2

По данным из задачи №1 построить столбиковую диаграмму. При наведении на столбоц он меняет цвет, а над ним появляется значение температуры в этот день. При убирании курсора подсветка исчезает. При необходимости можно переключить неделю в селекте. Использовать canvas.
