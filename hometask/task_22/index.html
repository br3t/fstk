Домашнее задание по jQuery

Задача №1

На странице располагается div шириной 600 пикселей залитый зеленым фоном. При клике по нему он делится на два div'а пополам. При клике на эти половинки вновь происходит деление и т.д. С вероятностью 25% при делении часть div'a превратится в неделимую, закрашенную в красный. Клики по неделимой части игнорируются. Написать скрипт, используя jQuery.

Задача №2

На странице располагается несколько div имеющих разный фон, есть кнопка для добавления новых в конец списка. Каждый div содержит кнопки: переместиться перед предыдущим блоком; перемститься после следующего блока; переместиться внутрь предыдущего блока; переместиться из родительского блока и стать после него. Если каколибо действие в какой-то момент недоступно - блокировать соответствующую кнопку. Написать скрипт, используя jQuery.
