Домашнее задание по теме HTML

Все задачи выполнить в одном файле, разделив заголовками второго уровня.

Задача №1

Сверстать список покупок, показанный на изображении.

Задача №2

Сверстать график дежурств, показанный на изображении. Не использовать CSS, можно использовать deprecated-атрибуты.

Задача №3

Сверстать форму регистрации, показанную на изображении. Не использовать CSS, для форматирования использовать только перенос строк. Имена полей произвольные. Форма должна отправляться в google.com методом post. В скрытом поле sessid должен передаваться произвольный идентификатор сессии.

Задача №4

Сверстать статью, показанную на изображении. Не использовать CSS. Использовать семантическую верстку. При наведении на подчеркнутое HTML должна появляться всплывающая подсказка.
