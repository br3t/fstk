Домашнее задание по HTML5. Часть 2

Задача №1

Написать вэб-приложение "Текстовый редактор". Экран приложения состоит из двух частей: списка черновиков и рабочей области. По клику по кнопке можно создать новый черновик; по клику на черновике он загружается в рабочую область для редактирования. Черновик сохраняется по клику по кнопке "Save", а также периодически в процессе редактирования. Рабочая область - простое textarea. Вместе с текстом черновик сохраняет дату создания и дату последнего редактирования. Список черновиков и сами черновики хранятся в localStorage.
