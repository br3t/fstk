Домашнее задание по jQueryUI

Задача №1

Создать страницу с формой загрузки резюме. В форме должны быть:

  • datepicker для выбора даты рождения
  • поле для выбора языков программирования с autocomplete
  • выбор личных качеств соискателя в виде sortable-списка от самой сильной к наименее сильной
  • динамическое добавление мест работы с информацией о позиции, компании и сроке работы (spinner)
  • диапазон ожидаемой зарплаты в виде slider

На странице также размещены tab'ы с информацией о компании и описанием вакансии.

Использовать jQueryUI с проивзольной темой. Библиотеку и ее файлы подключать с CDN.

Указания по выполнению задания

  • проект состоит из файлов HTML, CSS, JavaScript в репозитории
